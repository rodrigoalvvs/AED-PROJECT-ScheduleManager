bool Student::addClass ( const std::string &  ucCode,
const std::string &  classCode 
)

Register the student in a class within a specific course unit.

This method registers the student in a class of a course unit specified by the provided UC code and class code. The student can be registered in multiple course units, but cannot register in the same course unit more than once.

Parameters
ucCodeThe code of the course unit to register in (e.g., L.EICXXX).
classCodeThe code of the class to register in (e.g., XLEICXX).
Returns
True if the registration was successful, false if the student is already registered in the unit.
