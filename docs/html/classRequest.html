Initializes a new request object.

This constructor creates a new request object with the specified parameters, including the request type, student ID, items to be added, items to be removed, and a request ID.

Parameters
requestType_The type of the request (1: Add student to class, 2: Remove student from class, 3: Switch UC, 4: Switch class).
studentId_The unique identifier of the student associated with the request.
